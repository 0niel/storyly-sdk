# storyly-moments-react-native

The React Native plugin for Moments by Storyly

## Installation

```sh
npm install storyly-moments-react-native
```

## Usage

```js
import { multiply } from "storyly-moments-react-native";

// ...

const result = await multiply(3, 7);
```
